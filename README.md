# DDoS Эксперименты

Простое приложение для работы с экспериментами по DDoS-атакам.

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

Измените параметры подключения в `main.py`:
```python
DB_CONFIG = {
    'host': 'localhost',
    'port': 5432,
    'database': 'postgres',
    'user': 'postgres',
    'password': 'ваш_пароль'
}
```

## Запуск

```bash
python main.py
```

## Использование

1. **Создать базу** - создает схему и таблицы
2. **Внести данные** - открывает форму для добавления эксперимента
3. **Показать данные** - просмотр с фильтрами по типу атаки и дате
